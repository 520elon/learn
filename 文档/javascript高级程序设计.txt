第三章 基本概念
	1.  标识符必
		1. 须是字字母下划线或美元符开头，其他字符可以是字母，下划线，美元符或者数字。
		2. 推荐使用驼峰大小写格式。
		3. 保留着关键字不能用作标识符
	2. 变量

		1. js的变量是松散的，可以用来保存任意类型的数据。（未经初始化的变量会保存一个特殊值--undefined）
		2. var 定义的变量将成为定义该变量作用域中的局部变量，在变量退出该作用或后会被销毁。
		3. 如果省略了var 则变成了全局变量。在函数外部的任何地方都会被访问到
	3. 数据类型
		1. ECMAScript有五种简单的数据类型
			1. Undefined
				1. 只有一个值：Undefined。在使用var声明变量但未对其初始化的时候该变量的值就是Undefined。
			2. Null
				1. 第二个只有一个值的数据类型，特殊值为null，表示一个空对象指针。
				2. 如果定义的变量准备用来保存对象，最好将其赋值为null；
				3. undefined值派生自null，所以null == undefined   //true
			3. Boolean
				1. 有两个字面量：true  false
			4. Number
				1. 使用
			5. String
		2. 一种复杂的数据类型
			1. Object
				1. Object本质上是一种无序的明值对组成的。
				2. 不支持任何创建自定义类型的数据
	4. typeof
		1. typeof 操作符可以是变量也可以是数值字面量（在计算机科学中，字面量是用于表达源代码中一个固定值的表示法）
		2. 在调用 typeof null 会返回 “Object”，应为特殊值null被认为是一个空的对象引用
